# 03-project-management-app

A project management UI built with React and TypeScript. It is the third exercise in the [react-skill-lab](../../) monorepo.

## Scripts

From this directory:

| Command        | Description              |
| -------------- | ------------------------ |
| `pnpm dev`     | Start Vite dev server    |
| `pnpm build`   | Type-check and build     |
| `pnpm lint`    | Run ESLint               |
| `pnpm preview` | Preview production build |

From the monorepo root:

```bash
pnpm --filter 03-project-management-app dev
```

## Stack

- React 19
- TypeScript
- Vite
